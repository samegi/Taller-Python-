# Taller de Python – Estructuras y Fundamentos

**Autor:** Sara Mejia Giraldo  
**Fecha:** 08-02-2026

## Descripción
Repositorio con cuadernos de práctica de Python. Cada cuaderno incluye membrete, ejercicios resueltos, explicación de algoritmos, y conclusiones.

## Estructura
- 01_cadenas: ejercicios y manipulación de strings
- 02_tuplas: tuplas, desempaquetado y uso típico
- 03_listas: listas, métodos, slicing y casos prácticos
- 04_conjuntos: sets, operaciones y eliminación de duplicados
- 05_diccionarios: dicts, iteración y estructuras anidadas
- 06_condiciones: if/elif/else y operadores lógicos
- 07_bucles: for/while, break/continue y patrones
- 08_funciones: definición, parámetros, retorno, buenas prácticas
- 09_clases: POO básica (atributos, métodos, constructor)
- 10_bono: ejercicio integrador (opcional)

## Requisitos
- Python 3.x
- Jupyter Notebook (VSCode)

## Instrucciones de ejecución
1. Abrir el cuaderno correspondiente al tema.
2. Ejecutar las celdas en orden.
3. Revisar conclusiones al final.
